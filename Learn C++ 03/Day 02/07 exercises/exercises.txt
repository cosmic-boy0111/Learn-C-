1. Write a program to convert speed given in kilometers per hour to miles per hour.
   Sample Input:
   Input speed in Km/hr: 60

   Sample Output:
   Speed in miles per hour is: 37.2823

2. Write a program to convert the temperature given in Celsius to Fahrenheit.
   Sample Input:
   Input temperature in Celsius: 37.5

   Sample Output:
   Temperature in Fahrenheit is: 99.5

3. Write a program in C++ to calculate effective resistance of 3 resistors connected in parallel configuration.
   Sample Input:
   Input Resistance 1: 2
   Input Resistance 2: 5
   Input Resistance 3: 10

   Sample Output:
   Total Effective Resistance is 1.25 ohms.

4. Write a program to check if a given number is prime or not.
	Sample Input:
	Input a number to check prime or not: 13

	Sample Output:
	The entered number is a prime number.

5. Write a program to calculate the factorial of a number.
	Sample Input:
	Input a number to find the factorial: 5

	Sample output:
	The factorial of the given number is: 120

6. Write a program to calculate the Greatest Common Divisor (GCD) of two numbers.
	Sample Input:
	Input the first number: 25
	Input the second number: 15

	Sample Output:
	The Greatest Common Divisor is: 5

	Sample Input:
	Input a number: 1234

	Sample Output:

	Sample Input:
